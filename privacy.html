隱私權政策

最後更新日期：2026 年 7 月

本應用程式重視使用者的隱私。以下說明本應用程式對於資料的處理方式。

一、資料蒐集

本應用程式不會蒐集、儲存或分享任何使用者的個人資料。應用程式僅讀取並顯示 Discord 伺服器公開提供的 Widget 資訊（例如線上人數、頻道名稱等），這些資訊由 Discord 官方 API 直接提供，並非由本應用程式蒐集或儲存。

二、資料使用

由於本應用程式不蒐集任何使用者資料，因此也不涉及資料的使用、分析或第三方分享。

三、第三方服務

本應用程式的運作依賴 Discord 官方 API，相關資料的取得與使用皆遵循 Discord 本身的服務條款與隱私權政策，建議使用者一併參閱。

四、政策修改

本政策可能會不定期更新，異動後將於本頁面公告，請使用者自行留意。

五、聯絡方式

如對本政策有任何疑問，歡迎透過 GitHub 與開發者聯繫。
